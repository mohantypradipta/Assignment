# Frontend - Medical Triage SPA

React + TypeScript single-page application for medical triage assessment.

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

The app will be available at `http://localhost:3000`

## Build

```bash
npm run build
npm run preview
```

## Features

- 5-question symptom assessment
- Backward navigation with answer persistence
- Responsive mobile-first design
- Real-time validation
- Smooth animations and transitions
- Integration with backend API for assessment and booking
- Confirmation tracking with localStorage

## Architecture

- **State Management**: Zustand for global questionnaire state
- **Styling**: SCSS modules with CSS variables
- **HTTP Client**: Axios for API calls
- **Components**: Modular, reusable React components
